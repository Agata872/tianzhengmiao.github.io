GNN-based Precoding for Cell-free Massive MIMO

This project explores the use of Graph Neural Networks (GNNs) to design and fine-tune precoders in a cell-free massive MIMO system using real-world channel state information (CSI) gathered from our 6G testbed.

Background

Conventional linear precoding approaches struggle with distributed deployments and real CSI irregularities. We aim to learn flexible, robust precoding mappings using data-driven methods.

Methodology

The proposed approach models the network as a graph, where Access Points (APs) and Users are nodes. A multi-layer GNN architecture learns to output precoding weights based on node and edge features.

Results

  • Performance improves over traditional MRT and ZF in realistic deployments.
  • Transfer learning enables cross-scenario generalization with minimal fine-tuning.
